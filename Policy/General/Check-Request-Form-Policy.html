The Check Request Form must be filled out each time a check request is made. However, if the expense can be paid via credit card, the Check Request Form is not necessary. The Check Request Form is only for submitting paper checks.

  • Check requests will no longer be processed via Google Chat, text, telephone, verbal conversations, or email.

Cenvar Roofing Check Request Form

Filling out the Cenvar Roofing Check Request Form

Enter the Payee name

Enter the dollar amount

Select the purpose for the check request. If you do not see the appropriate purpose, select ‘Other’ and enter the purpose.

Enter the Job Number if applicable

Press the green “Submit” button

This form will be sent directly to the Cenvar Roofing Accounting Department to process the check request.
