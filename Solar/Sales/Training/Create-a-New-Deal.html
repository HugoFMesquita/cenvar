In the Deals then Sales Pipeline click the Create Deal orange box in the upper right corner. Complete all the information in the drop-down menu and then click Create deal.

  • Deal name
    • Format as: First Name, Last name@address, city, state zip code
  • Pipeline
    • Sales Pipeline
  • Deal Stage
    • Select the appropriate stage from the drop-down menu.
  • Sales Rep
    • Select the appropriate name from the drop-down menu.
  • First name
  • Last Name
  • Email
  • Phone Number
  • Street Address
  • City
  • State
  • Branch Location
    • Cenvar Branch
  • Zip Code
  • Lead Source
    • Choose from Drop-down menu.
      • If it is a Customer Referral 
        • On the left sidebar scroll down to Referral.
          • Under Referral select
            • Yes
            • No
        • Referrer Name
        • Enter the first and last name of the person who referred this deal.
        • Referrer Email
          • Enter the email address of the person who referred this deal.
        • Referrer Address
          • Enter the address, city, state, zip of the person who referred this deal.
        • Referrer Phone
          • Enter the phone number xxx-xxx-xxxx of the person who referred this deal.
      • If the customer is unable to provide this information leave a note for the sales specialists and ask them to collect the information at the appointment.
  • Associate deal with 
    • Contact
      • See if deal is in the drop-down menu and click in box for the appropriate deal name.
      • If the deal name is not present leave blank
  • Click the Create orange button.
  • In the About this deal section
    • Deal Owner
      • Select from drop-down menu (if applicable)
    • Sales Rep
      • Select from drop-down menu (if applicable)
    • Setter
      • Select the name from the drop-down menu.
    • Customer Name
      • Enter first and last name.
    • Deal Type
      • Select from drop-down menu.
      • Site Location
        • Enter the address of where the solar will be installed.
          • Street, City, State, Zip
      • Google Map Location
        • Copy the Site Address. 
        • Go to Google.
        • Paste the address.
        • Click the street view.
        • Copy the link from Google.
        • Paste into Google Map Location in Hubspot.
      • Click the Save Orange Button at bottom.

In the Appointment Info

  • Verify all the fields have been populated.
  • Shading
    • Select from the drop-down menu based on the Google view of their house.
  • Appointment Type
    • Select Virtual or In-Home
  • Appointment Date
    • Enter the date of the appointment.
  • Appointment Time
    • Enter the time of the appointment.
  • Aurora Trigger
    • Select Trigger from the drop-down menu.
  • Scroll down and click the Orange Save Button.

Add notes from the phone call.

  • In the center of your screen select Notes.
  • Click the Blue Create Note button to the right.
  • Enter notes pertaining to the call with the customer to pass along to the Sales Specialist.
  • Click the Save Note button.
