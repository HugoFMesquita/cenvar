Set up file in share drive

Go to Sales Rep’s Name Folder name on the Google drive.

  1. Open the Solar Proposal Folder
  2. Select File
  3. Select Make a copy.
  4. Rename the file as Sales Proposal_Contact Name and Address of contact.
    • If multiple proposals are created, name each of them according to differentiating information, such as the location of panels, number and type of panels, etc.

Change slides:

Slide 1 Name and address information

  1. Add the customer’s name and address.

Slide 5 

  1. Go to Aurora
  2. Click the Customer’s name under Project.
  3. Click the bar under Design.
  4. In the Site System box to the left click << to hide this box.
  5. Click the hand icon in the lower right corner, it will turn black.
  6. Move the house to the left of the screen and zoom in on the house.
  7. In the upper right corner select the V to the right of Size Production Savings.
  8. Take a screenshot of this image.
  9. Import this screenshot into the Sales Proposal.

Slide 6 Cash payment

  1. Fill in all numbers highlighted in yellow.
  2. Once you have completed all the info, unhighlight all yellow areas.
  3. If you use this slide, delete slides 7 and 8.

Slide 7 Financing 

  1. Fill in all numbers highlighted in yellow.
  2. Once you have completed all the info, unhighlight all yellow areas.
  3. If you use this slide, delete slides 6 and 8.

Slide 8 (is to be used when a customer would like to see the difference between cash and financing).

  1. Fill in all numbers highlighted in yellow.
  2. Once you have completed all the info, unhighlight all yellow areas.
  3. If you use this slide, delete slides 6 and 7.

Slide 9 SRECs

  1. Fill in the # of SRECs in bullet point 2 & bullet point 3. Unhighlight all yellow areas.

Save file and send to client

  1. Save file as a pdf under the Sales Rep’s Folder on the Google Solar Sales Drive.
  2. Email to the client as a pdf through HubSpot.
    1. When you email the proposal to the customer through HubSpot it will automatically show up in the Attachments sidebar on the right.
    2. If multiple proposals were created for the customer, save each proposal in HubSpot and create a new file name (Add a 1, 2, etc. at the end of the current file name). If you email them to the client through HubSpot they will show as an attachment in the right sidebar under Attachments.
    3. Add the link to the Chris Good video thanking them for the opportunity to provide them a proposal.
    4. If the attachment does not show up in HubSpot.
      1. Go to Attachments in the right-side bar.
      2. Click Add
      3. Click Upload New
      4. Choose the file to upload.
