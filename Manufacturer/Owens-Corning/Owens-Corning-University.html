Owens Corning University (OCU) is your direct connection to all things education. Users are able to access mini-modules for learning on the go, and a robust reference library.

Cenvar Roofing Project Coordinators are required to complete all OC University training modules within the first 4 weeks of their hire date AND again within month 6-8 after hire date.

**Setting up individual employee modules in the OC University:

  • The branch General Manager will create the new user account in Owens Corning Connect.

1. Open the Owens Corning Connect Website

2. Select “Settings” from the menu.

3. Select “User Management” from the menu or in the body.

4. To create a new user, press the pink “Add/Remove Users” button.

5. Fill in all fields:

  • Add or Remove contact: select “Add”
  • Portal Role: select “Sales & Marketing” roll.
  • Company Name: Cenvar Roofing
  • Member ID: enter the Cenvar Roofing email address of the user
  • Company City: enter the current branch location
  • First Name of Contact: enter the first name of the user
  • Last Name of Contact: enter the last name of the user
  • Email of Contact: enter the Cenvar Roofing email address of the user
  • Contacts Mobile Phone: enter the cell phone number of the user
  • Contacts Job Title: enter the title of the user
  • Requesters First & Last Name: enter the first & last name of the General Manager

6. Press the blue Submit button.

7. The following message will be displayed:

  • “Thank you for submitting your new user request! Your request will be processed within 48 business hours!”

8. The new user will receive an email once Owens Corning processes the new user into the system.

The user email will contain a link with instructions to log in for the first time, including creating a new password for their account.

The branch General Manager is responsible for monitoring the Project Coordinator Owens Corning University module progress and completion.

If you experience issues logging into your Owens Corning University account, please contact your branch General Manager for assistance.
