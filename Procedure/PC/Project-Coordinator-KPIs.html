• % Impressed: Goal: 80% / Minimum: 60%
  • % Likely To Recommended: Goal: 98% / Minimum: 96%
  • Google Reviews: Goal: 50% / Minimum: 20%

The Project Coordinator's KPIs will be measured using a rolling three-month average. If performance falls below the required minimum percentage, the Project Coordinator will enter a one-month probationary period. Failure to meet the minimum percentage by the end of probation will result in termination. Additionally, a Project Coordinator may not be placed on probation more than twice per year.
